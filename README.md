# 心流白板

## 📖 项目介绍

**心流白板** 是一个基于浏览器的本地优先（Local-First）个人效能管理工具。它不需要服务器，不需要注册账号，所有数据都存储在你自己的电脑上，绝对安全隐私。

这款应用结合了**艾森豪威尔矩阵（四象限法则）**、**周计划**、**长期目标**和**番茄工作法**，旨在帮助你更专注、更高效地管理日常生活和工作。它采用极简主义设计，支持暗色模式和纸感主题，让你沉浸在高效的心流体验中。

## ✨ 核心功能

*   **📅 四象限每日任务**：将任务按“重要/紧急”程度分类，清晰排列。
*   **🗓️ 周计划视图**：规划一周重点，未完成任务自动顺延。
*   **🎯 长期目标**：设定并追踪你的宏大目标，支持子目标拆解。
*   **🔁 日常/习惯任务**：创建可复用的日常任务模板，一键拖拽生成。
*   **🍅 专注计时器**：内置番茄钟和秒表，自动记录每个任务的投入时间。
*   **📥 GTD 收件箱**：快速捕捉想法，稍后整理归档。
*   **📂 本地存储**：利用现代浏览器的文件系统访问 API，直接读写你选定的本地文件夹，数据完全掌握在自己手中。
*   **📊 数据复盘**：自动生成每日/每周复盘摘要，支持 Markdown 编辑，提供可视化数据统计。
*   **🤖 AI 辅助（可选）**：支持连接 OpenAI 兼容接口（如 DeepSeek, Ollama 等），实现智能任务拆解和每日总结分析。

## 🚀 安装指南

由于本项目是一个纯前端应用，**无需安装任何复杂的依赖**（如 Node.js 或 Python 环境）。

1.  **下载代码**：
    *   点击页面右上角的 `Code` -> `Download ZIP` 下载压缩包并解压。
    *   或者使用 Git 克隆：`git clone https://github.com/your-username/infinity-flow.git`
2.  **运行应用**：
    *   直接双击文件夹中的 `index.html` 文件。
    *   推荐使用 **Chrome**、**Edge** 或其他基于 Chromium 的现代浏览器（以获得最佳的文件系统 API 支持）。

## 📖 使用方法

1.  **初始化工作区**：
    *   首次打开时，点击屏幕中央的“打开本地工作区”按钮。
    *   选择电脑上任意一个空文件夹（或专用文件夹），该文件夹将用于存储所有的 JSON 数据文件。
2.  **添加任务**：
    *   在四象限（Q1-Q4）下方的输入框输入任务并回车。
    *   或者在左侧边栏添加周任务、长期目标。
3.  **管理任务**：
    *   **拖拽**：支持在不同象限、周任务、收件箱之间自由拖拽任务。
    *   **计时**：点击任务卡片右侧的计时按钮开始专注。
    *   **完成**：拖动滑块至 100% 即可完成任务。
4.  **快捷键**：
    *   支持 `Ctrl + F` (或 `Cmd + F`) 呼出全局搜索。
